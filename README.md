# EcoRoute - Carbon Footprint Tracker

EcoRoute is an Android app that calculates CO₂ emissions for travel routes and helps users compare transport modes.

## Features
- Route carbon footprint calculator
- Transport emissions comparison
- Climate action savings calculator
- Public transport impact estimator

## APK Download
Download the latest APK from the /apk folder.

## Built With
- OpenStreetMap
- OSRM Routing
- Nominatim Geocoding
